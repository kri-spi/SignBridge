# SignBridge

## Project Description
SignBridge is a React Native application designed to facilitate seamless communication through gesture recognition and text conversion.

## Installation Instructions
1. Clone the repository: `git clone https://github.com/kri-spi/SignBridge.git`
2. Navigate to the project directory: `cd SignBridge`
3. Install dependencies: `npm install`

## Running the Frontend
To run the frontend application:
1. Navigate to the `SignBridge` directory: `cd SignBridge`
2. Start the Expo app: `npx expo start`

## Running the Backend
To run the backend server:
1. Navigate to the `backend` directory: `cd backend`
2. Start the server: `python server.py`

## Running on the Web

To run the simulator on the web, use the following command:

```bash
npx expo start --webs
```

This application is optimized for the web, providing a seamless experience across devices.

## Usage
* On iOS: Scan the QR code with the Camera app.
* On Android: Scan the QR code with the Expo Go app.

## Features
- Gesture recognition for text input
- Real-time text conversion
- User-friendly interface
